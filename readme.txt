== Description ==

Description: FreeNews Magazine theme is an amazing modern WordPress theme. You can choose to build professional websites. It is easy to customize for all kinds of news, newspaper, magazine, publishing, blog or review sites. It also include all major aspects like responsive, performance, cross-browser compatible, SEO ready and supports RTL. It is ready to promotion with social media icons to reach maximum target audience. Responsive slider impress your customers with lively eye-catching images right on your banner section. Get free Supports at https://tickets.themefreesia.com/, View demos at https://demo.themefreesia.com/freenews and documentation at https://themefreesia.com/theme-instruction/freenews

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How can i add social icons? =
In the Dashboard, select Appearance > Menus.
Select “create a new menu” at the top of the page.
Enter a name for your new menu in the Menu Name box.
Click the Create Menu button.
Now, choose the Custom Url from left side with social icon url and name. Click on add menu
Click save menu after creating social icons

Please refer freenews documentation at https://themefreesia.com/theme-instruction/freenews

 Now all your files and settings has been imported. Now you just need to setup your menu and front page.

 Setup Home Page:
 	- How to setup Home Template?
		Log in to your WordPress Administration Panel (Dashboard).

		Go To Dashboard > Pages > Add New and Give it a title.

		In the page editor, there is a box on the right called “Page Attributes“. From this box,select "Home Template" and save the page.

		Now your Home Template is setup. To setup Banner and Highlighted Category. Read About Main Banner & Highlighted Category Left/ Right Documentation

	- How to setup Header Banner ?

		Log in to your WordPress Administration Panel (Dashboard).
		Go to dashboard and navigate to Appearance > Widgets > Drag Image Widgets to Header Banner Sidebar

	- How to setup Advertise Area ?

		Log in to your WordPress Administration Panel (Dashboard).
		Go to dashboard and navigate to Appearance > Widgets > Drag Image Widgets to Advertise Area Sidebar

	- Navigate to Dashboard > Reading > Click on ( A static page ) from Your homepage displays
 	- Select Homepage as Home and Postpage as Blog

	- How to setup FreeNews Template Left, Main and Right Sidebar section ?

		Log in to your WordPress Administration Panel (Dashboard).
		Go to Posts > Categories. To add a category and create it.
		Then Click the ‘Posts’ tab to create a new post
		Click the ‘Add New’ sub-tab.
		Start filling in the blanks: enter your post title in the upper field, and enter your post body content in the main post editing box below it. On the right bottom of posts add featured image
		Repeat above process multiple times to display in Left/ Right Highlighted Category section.
		As needed, select a category just you created for both Left, Main and Right Sidebar section
		Now setup Widgets on your Left, Main and Right Sidebar section

		Log in to your WordPress Administration Panel (Dashboard).
		Go to dashboard and navigate to Appearance > Widgets > Drag T-Spiral: Blog Category Posts, T-Spiral: Category Slide, T-Spiral: Latest Posts and T-Spiral: List Category Posts widgets
